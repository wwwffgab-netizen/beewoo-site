# BeeWoo 应聘网站

饼干车库 —— 主播 + AI视觉应聘专用页。

## 本地预览

双击 `index.html` 在浏览器里打开。

## 上线（选一个，都免费）

### 方案一：GitHub Pages（推荐）
1. 注册 GitHub 账号
2. 新建仓库，把 `index.html` 传上去
3. Settings → Pages → 选 main 分支 → 保存
4. 几分钟后得到链接：`https://你的用户名.github.io/仓库名`

### 方案二：Vercel
1. 注册 vercel.com
2. 把 `beeWoo-site` 文件夹拖进去
3. 自动部署，生成链接

### 方案三：直接发文件
把 `index.html` 发给饼干，他浏览器打开就能看。

## 你需要改的地方

1. **联系方式**（搜索"替换为"）→ 填你的真实微信、电话
2. **作品区**（搜索"img-placeholder"）→ 把占位符换成你做好的 AI 图
3. **微信号/电话** → 在 Contact 区块

## 上线后

把链接发给饼干，附一句话：
> "饼干哥，这是我专门为应聘做的页面。你看看。"
